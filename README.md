# Srping Eureka Example
    Eureka server
    Eureka client(Student Service)

# Usage

Eureka Server
```
cd eureka-server
mvn spring-boot:run
```

Euraka Client(Student Service)
```
cd eureka-client
mvn spring-boot:run
```

### Output
Eureka server: http://localhost:8761/
Applications: http://localhost:8080/applications
